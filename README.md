## Getting Started

### To run the project:

1. Clone the repo
   ```sh
   git clone https://github.com/hzeghari/nano-quotes.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run npm start
   ```sh
   npm run dev
   ```
